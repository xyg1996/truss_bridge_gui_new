***********************
Application preferences
***********************

The preferences of the application are accessible via the menu
*File > Preferences* or the keyboard shortcut *Ctrl+P*.

Preferences
===========

The *Preferences* dialog allows modifying different parameters of the application.

.. note::
    The language of the application can be modified in the main preferences of
    SALOME (a restart will be needed).

General preferences
-------------------

This page makes it possible to define general parameters of the *AsterStudy*
application.

The user can also choose whether to use the default **version of Code_Aster**
for the creation of new case, or ask to choose a version during the creation.

Finally some options allow the user to configure the application by choosing
the **Workspace tab pages position**, use or not **business-oriented translations**
by default etc.
