Bienvenue chez AsterStudy¶

AsterStudy est un module SALOME qui fournit une interface utilisateur graphique permettant de mettre en place des études de cas pour Code_Aster.

Elle permet également de lancer les calculs, de suivre l’avancement de ces calculs et d’accéder aux résultats.
