Fichier de configuration¶

Le fichier de configuration est un fichier optionnel au format INI qui décrit le contenu de l”assistant de calcul :

Option Description Valeurs par défaut
declaration Nom du fichier de déclaration. declaration.json
wizard Nom du fichier template. wizard.py
template Nom du fichier assistant. template.comm
title Titre de l’assistant de calcul. Ce titre est affiché dans le menu de l’application. none
ordre Ordre d’apparition dans le menu (type int attendu).  
translator Nom du contexte de traduction. Normalement, cette valeur correspond à l’attribut class de l’assistant dans le fichier de déclaration. « Wizard »

Le fichier de configuration n’est pas obligatoire. Il est nécessaire seulement dans le cas où les fichiers par défaut ne sont pas utilisés.

Dans le futur, le fichier de configuration pourra être utilisé pour définir plusieurs assistants dans le même répertoire.

Exemple :

[General]
declaration = decl.json
wizard = impl.py
template = elastic.comm
title = Isotropic Linear Elasticity
translator = IsotropicLinearElasticity
order = 2
