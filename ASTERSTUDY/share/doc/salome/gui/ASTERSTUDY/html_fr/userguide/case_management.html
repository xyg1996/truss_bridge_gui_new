Gestion des cas¶

Sauvegarde d’un cas¶

Il est possible de sauvegarder un cas sans l’exécuter afin de conserver des versions intermédiaires du cas et de pouvoir les reprendre.

Pour cela, il faut le cas, soit dans la vue Historique ou dans la vue Cas, et sélectionner Opération > Sauvegarder ou l’entrée Sauvegarder du menu contextuel.

Sauvegarder un cas

Les cas sauvegardés sont rangés dans un sous-dossier séparé dans la vue Historique.

Organisation des cas dans la vue Historique

Supprimer une exécution¶

Il est possible de supprimer les résultats d’une exécution (à condition que ses résultats ne soient pas utilisés par d’autres exécutions).

Pour cela, il faut sélectionner l’exécution à supprimer, puis le bouton Supprimer de la barre d’outils, le menu Editer > Supprimer ou l’entrée Supprimer du menu contextuel.

Examiner un calcul¶

L’utilisateur peut voir le contenu d’un cas exécuté. Pour cela il faut sélectionner le cas dans la vue historique, puis l’entrée Vue du menu contextuel. Le détail du cas peut alors être consulté en mode lecture seule dans la vue Cas.

Vue en lecture seule

Cette fonctionnalité est utile car elle permet de consulter le détail des commandes composant les étapes du cas. En effet, entre deux calculs l’utilisateur a la possibilité de modifier les paramètres des commandes, d’ajouter ou supprimer des commandes sur les étapes. Les commandes exécutées lors d’une exécution peuvent donc être différentes de celles du cas courant.

Ajouter une description¶

L’utilisateur peut ajouter une description à chacune des exécutions ou des sauvegardes. Cela peut être fait via le menu Editer la description dans le menu Opérations ou via le menu contextuel.

Créer un cas¶

Le cas initial vide est automatiquement ajouté à l’étude lors de sa création. Cependant, il est possible de supprimer tout le contenu du Cas courant et de créer un nouveau cas vide. Pour cela utilisez l’opération Nouveau cas.

Exporter un cas¶

Le contenu du cas d’étude peut être exporté dans les fichiers avec l’opération Exporter cas disponible dans le menu principal Opérations ou via le menu contextuel. Cette opération crée un fichier .export et met les étapes dans des fichiers .com?, portant le nom de l’étape, à proximité. Les fichiers de données d’entrée sont placés près du fichier .export.

Importer un cas¶

Le contenu de l’étude peut être importée depuis le fichier .export. Utiliser le sous-menu Importer un cas du menu Opérations pour cela.
