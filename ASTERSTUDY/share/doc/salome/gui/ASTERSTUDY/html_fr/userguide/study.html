Gestion de l’étude¶

La gestion de l’étude se fait via le menu Fichier, cette fonctionnalité est héritée de SALOME. Le terme étude signifie document pour la plate-forme SALOME.

L’utilisateur peut :

  • Créer une nouvelle étude, en

    • Cliquant l’icône Nouveau document de la barre d’outils,
    • Utilisant le menu principal Fichier > Nouveau,
    • Utilisant le raccourci clavier Ctr+N.

  • Ouvrir une étude existante, en

    • Cliquant l’icône Ouvrir document de la barre d’outils,
    • Utilisant le menu principal Fichier > Ouvrir,
    • Utilisant le raccourci clavier Ctrl+O.

  • Sauver l’étude dans un fichier, en

    • Cliquant l’icône Enregistrer document de la barre d’outils,
    • Utilisant le menu principal Fichier > Enregistrer ou Fichier > Enregistrer sous…,
    • Utilisant les raccourcis clavier Ctrl+S ou Ctrl+Shift+S.

Le fichier principal de l”étude est le fichier .hdf. À côté de ce fichier .hdf, un dossier de même nom et se terminant par _Files est créé pour enregistrer d’autres informations. Ce répertoire contient aussi un fichier texte nommé data.ajs qui peut être utilisé en cas de problème par le Mode de récupération (voir En cas d’erreur de l’application - Mode de récupération).
