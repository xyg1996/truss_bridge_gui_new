Configuration file¶

Configuration file is an optional file in INI format, describing the content of the Calculation Assistant, namely:

Option Description Default value
declaration Name of declaration file. declaration.json
wizard Name of template file. wizard.py
template Name of wizard file. template.comm
title Calculation assistant’s title. This value is shown in application’s menu. none
order Order in the application’s menu (int expected).  
translator Name of the resource translation context. Normally, this value corresponds to the class attribute of the wizard in the declaration file. “Wizard”

The configuration file is not mandatory; it is only needed in case if non-default names of assistant’s files are used.

Also, in future configuration file might be used to allow storing several assistants in the same directory.

Example:

[General]
declaration = decl.json
wizard = impl.py
template = elastic.comm
title = Isotropic Linear Elasticity
translator = IsotropicLinearElasticity
order = 2
